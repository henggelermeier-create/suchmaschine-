# Kauvio – AI-first Rebuild Bundle

Dieses Paket ist auf deinen Wunsch zugeschnitten:

- **voll stärker auf KI ausgerichtet**
- **großer Umbau statt Mini-Fixes**
- **Startseite minimal wie eine Suchmaschine**
- **Schweiz / CHF / Live-KI-Suche klar im Fokus**

## Enthalten

### Frontend
- neue minimalistische Startseite
- große KAUVIO-Wordmark
- sauberer Suchfokus
- normale Suche + Live KI Suche
- kompakter, ruhiger Resultatstil

### Backend / AI
- echter `ai_service`
- AI-Extraktion für Produktseiten
- Ranking- und Matching-Endpunkte als Grundlage
- AI-first Open-Web-Discovery

## Anwendung

1. vorhandene Dateien im Repo mit den neuen Versionen ersetzen
2. Integrationsschritte aus `docs/PATCH_NOTES.md` anwenden
3. Deployment neu bauen
4. `OPENAI_API_KEY` setzen
5. Live-Suche mit echten Produkten testen
